# Chess Diagram Editor

**Chess Diagram Editor** — мощный редактор шахматных диаграмм на Python с PyQt6.  

Позволяет:  
- Рисовать шахматную доску 8×8 с координатами a–h / 1–8  
- Загружать свои фигуры (PNG/SVG) и устанавливать их на клетки  
- Drag & drop фигур и удаление ПКМ  
- Настраивать размер клеток и фигур  
- Задавать цвет или текстуры клеток (для белых и чёрных)  
- Настраивать цвет и толщину рамки доски  
- Рабочее пространство вокруг доски  
- Стартовая позиция и очистка доски  
- Экспорт диаграмм PNG высокого качества (300/600 DPI)
